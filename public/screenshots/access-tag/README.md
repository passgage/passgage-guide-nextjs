# Access Tag Installation Photos

Bu klasör Access Tag fiziksel kurulum rehberi için fotoğrafları içerir.

## Gerekli Fotoğraflar

### Step 1: Package Contents
- [ ] Access Tag kutu içeriği (üstten görünüm)
- [ ] Access Tag etiketi close-up
- [ ] Montaj donanımları (vida, dübel, yapışkan)
- [ ] Kullanım kılavuzu ve belgeler
- [ ] Ürün boyutları ölçüm fotoğrafı

### Step 2: Location Planning
- [ ] İdeal yerleşim yüksekliği görseli (1.2m-1.5m)
- [ ] İyi vs kötü yerleşim örnekleri
- [ ] Aydınlatma önemi (iyi ışıklandırılmış vs karanlık alan)
- [ ] İç mekan örnek montaj noktası
- [ ] Dış mekan örnek montaj noktası
- [ ] Kullanıcı yaklaşma açısı görseli

### Step 3: Installation
- [ ] **Yapışkan montaj süreci** (4 adım):
  - Yüzey temizleme
  - Koruyucu film çıkarma
  - Tag yerleştirme
  - Basınç uygulama
- [ ] **Vida montaj süreci** (4 adım):
  - Montaj noktalarını işaretleme
  - Pilot delik açma
  - Dübel yerleştirme
  - Vida sıkma
- [ ] **Manyetik montaj** (2 adım):
  - Manyetik taban montajı
  - Tag yerleştirme
- [ ] Seviye ölçer kullanımı
- [ ] Nihai montaj görünümü (yakın plan)
- [ ] Nihai montaj görünümü (uzak plan)

### Step 4: Configuration
- [ ] QR kod tarama örneği (telefon ile)
- [ ] Admin panel screenshot (tag ekleme ekranı)
- [ ] Admin panel screenshot (lokasyon atama)
- [ ] Başarılı yapılandırma mesajı
- [ ] QR kod test taraması

### Step 5: Maintenance
- [ ] Temizlik işlemi (yumuşak bez ile)
- [ ] Solmuş QR kod örneği (değiştirilmeli)
- [ ] İyi durumda QR kod örneği
- [ ] Montaj kontrolü yapılırken
- [ ] Hava koşulları hasarı örneği
- [ ] Yedek tag stoğu örneği

## Teknik Özellikler

- **Boyut**: Minimum 1920x1080px (landscape) veya 1080x1920px (portrait)
- **Format**: JPG (fotoğraflar için)
- **Boyut limiti**: Maksimum 800KB per photo
- **Kalite**: Yüksek çözünürlük, net odak
- **Aydınlatma**: Doğal ışık veya iyi aydınlatılmış ortam
- **Arka plan**: Temiz, dikkat dağıtmayan

## Çekim İpuçları

1. **Consistency**: Tüm fotoğraflar benzer açı ve aydınlatmada çekilmeli
2. **Hands in frame**: Montaj adımlarında eller görünmeli (işlem nasıl yapılıyor göstermek için)
3. **Before/After**: Montaj öncesi ve sonrası karşılaştırma fotoğrafları
4. **Close-ups**: Önemli detaylar için yakın çekim
5. **Wide shots**: Genel yerleşim için uzak çekim
6. **Real environment**: Gerçek iş ortamında çekim (sahte değil)

## Önemli Notlar

- Tüm montaj yöntemleri için foto serisi gerekli
- Gerçek Access Tag ürünü kullanılmalı
- Adım adım fotoğraflar kullanıcı için çok önemli
- QR kod netliği kritik - bulanık olmamalı
