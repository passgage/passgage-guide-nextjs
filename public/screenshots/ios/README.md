# iOS Screenshots

Bu klasör iOS rehberi için screenshot'ları içerir.

## Gerekli Screenshot'lar

### Step 1: Download
- [ ] App Store'da Passgage uygulaması görünümü
- [ ] İndirme butonu vurgulanmış

### Step 2: Permissions
- [ ] iOS Settings → Privacy → Location Services ekranı
- [ ] iOS Settings → Privacy → Camera ekranı
- [ ] iOS Settings → Notifications → Passgage ekranı
- [ ] Passgage uygulama izin isteme pop-up'ları

### Step 3: Login
- [ ] Passgage giriş ekranı
- [ ] Face ID/Touch ID onay ekranı
- [ ] Başarılı giriş sonrası ana ekran

### Step 4: Check-in Methods
- [ ] NFC check-in yapma ekranı
- [ ] QR kod tarama ekranı
- [ ] Başarılı check-in mesajı

### Step 5: NFC Setup
- [ ] (İsteğe bağlı - iOS'ta otomatik olduğu için gerekli değil)

### Step 6: Troubleshooting
- [ ] Safari location izni hatası ekranı
- [ ] GPS kalibrasyon ekranı
- [ ] Airplane mode açma/kapama görseli

## Teknik Özellikler

- **Boyut**: Minimum 750x1334px (iPhone 8/SE boyutu)
- **Format**: PNG veya JPG
- **Boyut limiti**: Maksimum 500KB per screenshot
- **Dil**: Türkçe iOS arayüzü
- **Device**: iPhone (modern model - notch veya Dynamic Island ile)
